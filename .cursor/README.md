# Cursor 配置文件说明

本目录包含了 Sprout 项目的 Cursor AI 编辑器配置文件，用于提升开发体验和代码质量。

## 文件说明

### 📋 `.cursor/rules`

**编码规范配置文件**

- 定义了项目的命名规范和代码风格
- 包含 TypeScript、Vue、UnoCSS 的使用规范
- 提供了文件组织和注释规范

### 📚 `.cursor/docs`

**项目文档配置文件**

- 详细的项目结构说明
- 开发指南和部署说明
- 技术特性和使用方法介绍

### 🤖 `.cursor/prompts`

**AI 提示词配置文件**

- 为 Cursor AI 提供项目上下文
- 定义了开发指导原则
- 包含常用的代码模板和模式

## 配置效果

有了这些配置文件，Cursor AI 将能够：

✅ **理解项目技术栈**：Nuxt.js + Vue 3 + TypeScript + UnoCSS
✅ **遵循命名规范**：自动使用正确的命名约定
✅ **生成符合规范的代码**：遵循项目的代码风格
✅ **提供上下文感知的建议**：基于项目结构给出合适的建议
✅ **优化开发工作流**：提供项目特定的开发指导

## 使用建议

1. **新团队成员**：阅读这些配置文件快速了解项目规范
2. **代码审查**：以这些规范为标准进行代码审查
3. **AI 辅助开发**：让 Cursor AI 基于这些配置提供更精准的代码建议
4. **持续更新**：随着项目发展及时更新配置文件

## 自定义配置

如需修改配置，请编辑对应的配置文件：

- 修改编码规范 → 编辑 `rules`
- 更新项目文档 → 编辑 `docs`
- 调整 AI 行为 → 编辑 `prompts`

---

_这些配置文件将帮助团队保持一致的代码风格，提高开发效率和代码质量。_
